# Project Brief: Portfolio Website

## Project Overview
Personal portfolio website for Harshit Chaturvedi, showcasing professional experience, education, and transition from Frontend Engineer to Product Manager.

## Project Goals
1. Present professional profile and experience effectively
2. Showcase technical skills and business acumen
3. Attract opportunities in Associate Product Management or Project Management
4. Demonstrate design aesthetic and modern web development capabilities

## Target Audience
- Recruiters and hiring managers
- Potential employers in Product Management
- Professional network connections
- Students and peers in HC BYTES community

## Current State
- Single-page portfolio website
- Built with HTML, Tailwind CSS, and vanilla JavaScript
- Fully responsive design
- Animated elements and modern UI patterns

## Key Features
- Hero section with animated gradient text
- About section highlighting technical PM capabilities
- Professional journey timeline
- Research & projects showcase
- Education credentials and certifications
- Contact section with social media links
- Animated background particles
- Smooth scroll navigation

## Technical Stack
- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Google Fonts (Inter)
- Hosted as static site

## Success Metrics
- Professional presentation of credentials
- Clear value proposition for PM roles
- Easy navigation and contact methods
- Fast load times and smooth animations
- Mobile-friendly experience
